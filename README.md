# Nora Çiçekçilik 🌸

Flask ile geliştirilmiş, modern ve mobil uyumlu bir çiçekçi web sitesi.

## Özellikler
- Ürün listeleme, kategori filtreleme
- Sepet yönetimi ve sipariş oluşturma
- İndirim kodu sistemi
- Sipariş takip
- Yönetim paneli (ürün, sipariş, kupon yönetimi)

## Kurulum
1. `pip install flask`
2. `python app.py`
3. Tarayıcıda `http://localhost:5000` adresini aç.

## Giriş Bilgileri (Panel)
- Kullanıcı adı: `NORA`
- Şifre: `nora2026`
