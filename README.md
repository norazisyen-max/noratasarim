# Nora Çiçekçilik 🌸

Flask ile geliştirilmiş, modern ve mobil uyumlu bir çiçekçi web sitesi.

## Özellikler
- Ürün listeleme, kategori filtreleme
- Sepet yönetimi ve sipariş oluşturma
- İndirim kodu sistemi
- Sipariş takip ve detay görüntüleme
- Yönetim paneli (ürün, sipariş, kupon, iletişim yönetimi)
- Stok takibi ve uyarısı
- Son 7 gün sipariş grafiği
- WhatsApp ile iletişim

## Kurulum
1. Projeyi klonlayın
2. `pip install flask` ile bağımlılıkları yükleyin
3. `python app.py` ile çalıştırın
4. `http://localhost:5000` adresini ziyaret edin
